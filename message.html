This example is a simple starting point to build hands-on botanical dissections and anatomy lessons within the web browser. For more information on various defensive structures and their plants' anatomy, please see this North Carolina Extension resource

PC & Mac Keyboard Controls

D Dissect and reset the plant
Q Highlight parts of the plant while dissected
W Reset the size of the selected organ
E Expand the selected organ
R Reduce the selected organ
G Grow information card
V Read selected organ information
Enter Select highlighted organ
